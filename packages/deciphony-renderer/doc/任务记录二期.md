# 功能

插件将分为

+ 标准谱
+ 鼓谱 二期或三期实现
+ 简谱
+ 吉他谱 二期实现
+ 贝斯谱
+ 尤克里里谱 三期实现

之前的分为标准，简谱，四线，六线的提议失效，因为不同谱子差别还是大
插件将永远不会实现一个复谱表多个谱子类型的提议。

musicScore的type将决定一切，musicScoreType会提示出目前使用的是鼓谱，五线谱，简谱，吉他谱等等，提示使用何种编辑或转换函数

## 吉他谱

吉他谱没有调号，谱号。只有拍号
暂时不考虑对齐线

+ ✅️六线小节皮肤
+ ✅️音符,休止符，符干，符尾
+ ✅️和弦（音符的）
+ ✅️附点
+ ✅️符尾连接
+ ✅️琶音，琶音有独立的，和带音符的两种
+ ✅️扫弦音符，扫弦有独立的，和带音符的两种
+ ✅️连音线（击弦H，滑弦S，勾弦P，击勾弦HP）
+ ✅️和弦符号（音符附属）
+ ✅️拍弦，方框框住 slap
+ ✅️推弦(一个曲线) bend

+ ✅️泛音，用<>框柱,自然泛音,人工泛音 Harmonic 人工泛音作为附属符号存在
+ 倚音 graceNote， 吉他谱的倚音符号不需要缩小，但是下面要有倚音的底座。吉他谱的倚音不会渲染符干和符尾。即使有chronaxie属性
  // 剩下都是附属符号 需要在engine里对不同谱表进行配置
+ 切音(制音)，箭头上的点 Staccato
+ 滑音（滑弦），无头，无尾滑音 Slide
+ 颤音（揉弦，还有一种，vb这个先不管），横着的波浪线 Vibrato
+ 自由延长，附属符号 Fermata
+ 弹片上拨，附属符号 Upstroke
+ 弹片下拨，附属符号 Downstroke
+ 点弦，T, 附符属符号 Tapping
+ 重音,>,附属号 Accent
+ 震音，三个斜线，附属符号 Tremolo
+ 闷音 Palm mute

如果还有时间，可尝试鼓谱
