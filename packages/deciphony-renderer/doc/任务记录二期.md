+ 标准谱
    + 五线谱
    + 鼓谱
+ 简谱
    + 简谱
+ 六线谱
    + 吉他谱
+ 四线谱
    + 贝斯谱
    + 尤克里里

插件将永远不会实现一个复谱表多个谱子类型的提议。
musicScore的type将决定一切，musicScoreType会提示出目前使用的是鼓谱，五线谱，简谱，吉他谱等等，提示使用何种编辑或转换函数

## 吉他谱

吉他谱没有调号，谱号。只有拍号
暂时不考虑对齐线

+ 六线小节皮肤
+ 音符,休止符，符干，符尾
+ 和弦（音符的），
+ 泛音，用<>框柱,自然泛音,人工泛音
+ 扫弦音符，扫弦有独立的，和带音符的两种
+ 琶音，琶音有独立的，和带音符的两种
+ 轮扫
+ 附点
+ 和弦符号（音符附属）
+ 符尾连接
+ 连音线（击弦H，滑弦S，勾弦P，击勾弦HP）
+ 切音(制音)，箭头上的点
+ 滑音，无头，无尾滑音
+ 颤音（揉弦，还有一种，vb这个先不管），横着的波浪线
+ 拍弦，方框框住
+ 推弦(一个曲线)
+ 自由延长，附属符号
+ 弹片上拨，附属符号
+ 弹片下拨，附属符号
+ 点弦，T, 附属符号
+ 重音,>,附属符号
+ 震音，三个斜线，附属符号
+ 倚音
