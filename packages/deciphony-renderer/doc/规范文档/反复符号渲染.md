# 反复符号渲染

反复符号（不含反复小节线 `StartRepeat_barline` / `EndRepeat_barline` 与双小节附属 `volta`）挂在 `Measure` 上，分**小节前**与**小节末**两类，各最多一个。

## 数据结构

```typescript
measure.startRepeat?: { id, type: MeasureStartRepeatEnum, relativeX?, relativeY?, ... }
measure.endRepeat?: { id, type: MeasureEndRepeatEnum, relativeX?, relativeY?, ... }
```

| 字段 | 枚举 | 位置 |
|------|------|------|
| `startRepeat` | `Coda` / `Segno` | 小节前 |
| `endRepeat` | `To_coda` / `DC` / `DS` / `Fine` / `DC_al_fine` / `DC_al_coda` / `DS_al_fine` / `DS_al_coda` | 小节末 |

前后可同时存在（例如小节前 `Coda` + 小节末 `DC`），同侧互斥。

## 渲染入口

`render/repeat/renderMeasureRepeat.ts`，在 `musicScoreToVDomImpl` 中于 `renderSymbol` 之后、`processBeam` 之前调用。线谱与简谱共用同一套逻辑，皮肤 key 相同（`repeat_*`）。

## 水平位置

设小节左上角为 `(measureX, measureY)`，宽 `measureWidth`，符号皮肤宽 `w`：

**小节前（`repeat_f`）— Coda / Segno**

- 符号几何中心 x 对齐小节左缘：`x = measureX - w / 2`
- 再加 `startRepeat.relativeX`

**小节末（`repeat_b`）— 文字类与 To Coda**

- 符号右缘对齐小节右缘：`x + w = measureX + measureWidth`，即 `x = measureX + measureWidth - w`
- 皮肤内文字 `text-anchor="start"`，从左向右排
- 再加 `endRepeat.relativeX`

## 垂直位置

默认在小节顶边上方：

```
y = measureY - skinH - REPEAT_MARK_ABOVE_RATIO * measureHeight + relativeY
```

`REPEAT_MARK_ABOVE_RATIO = 1/8`（见 `renderMeasureRepeat.ts`）。

## 皮肤

| skinKey | 内容 | 尺寸 |
|---------|------|------|
| `repeat_coda` | 30×30 占位 rect | 30×30 |
| `repeat_to_coda` | 30×30 占位 rect | 30×30 |
| `repeat_segno` | `<text>` Segno（居中） | 40×18 |
| `repeat_dc` … `repeat_ds_al_coda` | `<text>` 对应英文标注（左对齐） | 见 `defaultSkin.ts` |

Coda / To Coda 占位框后续可替换为正式 SVG path。

## VDom

| tag | 说明 |
|-----|------|
| `repeat_f` | 小节前反复 |
| `repeat_b` | 小节末反复 |

`zIndex = 1200`，与小节线、谱号同级。
