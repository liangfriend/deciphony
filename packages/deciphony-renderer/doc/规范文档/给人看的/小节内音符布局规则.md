符号可用宽度 = 小节宽度 - 前置符号宽度 - 后置符号宽度

### 普通模式

比如： | 5 - 5 5 |
可以想象成把 符号可用宽度分成等同于小节总时值数量的小格
这里总时值是256
第一个二分音符是从第0格开始的，第二个四分音符从第128格开始，第三个四分音符从第192格开始

会记录出一个widthRatio表，
{
0:默认值+音符widthRatio
128:默认值+音符widthRatio
192:默认值+音符widthRatio
}
总ratio = 上述表里所有值相加
每一个音符的宽度就是：
(默认值+音符widthRatio) / 总ratio * 符号可用宽度

在这个宽度范围内，简谱居左对齐，线谱居中对齐

简谱的皮肤widthRatio，在二分音符和四分音符的情况下，会给加时线复制一份
比如 皮肤widthRatio是x, 其他附属符号widthRatio是y, 附点widthRatio是z
#5 - - -. 虽然只有一个全音符，但是记录的widthRatio的表格却是
{
0:默认值+x+y
128:默认值+x
256:默认值+x+z
}

### 连谱模式

widthRatio表会记录同列小节的widthRatio
{
0:avg(小节1(默认值+音符widthRatio) + 小节2(默认值+音符widthRatio))
128:avg(小节1(默认值+音符widthRatio) + 小节2(默认值+音符widthRatio))
256:avg(小节1(默认值+音符widthRatio) + 小节2(默认值+音符widthRatio))
}
