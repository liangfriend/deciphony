五线谱编辑功能

+ 曲谱末尾添加复谱表按钮
+ 选中复谱表
    + 向前、向后添加复谱表
    + 添加单谱表
    + 删除
+ 选中单谱表
    + 向前、向后添加单谱表
    + 添加小节
    + 删除
+ 选中小节
    + 向前、向后添加小节
    + 添加音符、休止符
        + 音符前、音符上（多声部）、音符后
    + 添加volta
    + 添加调号
    + 添加拍号
    + 添加谱号 首行和其他行有差异
    + 添加小节线
    + 删除
+ 选中音符头
    + 上下拖拽更改音符region
    + 更改音符时值
    + 更改beamType，这个可能有点复杂，涉及到同步修改音符组内音符的direction
    + 添加、更改变音号
    + 添加、更改附点
    + 删除
+ 添加连音线
    + 选中音符的状态，选中的音符显示起始音符，再点击一个音符后添加连音线
+ 选中连音线
    + 拖拽ui
    + 删除
+ 选中变音号，附点
    + 删除
+ 选中volta
    + 拖拽ui
    + 删除
