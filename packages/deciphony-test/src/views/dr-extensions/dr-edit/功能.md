## 五线谱编辑功能

### 持续选中的状态

+ 音符 or 休止符, 默认音符
+ 全 二分 四分 八分 16分 32分 64分 128分 默认四分
+

### 曲谱上的添加功能

+ 曲谱末尾（g-d插槽）添加复谱表按钮，始终存在
+ hover, 鼠标移动到任意曲谱元素上都会加上一个特殊的高亮filter, 移出后取消，进入进出过程有0.2s动画
+ 选中小节，进入音符添加模式
    + 鼠标移动到某一范围可以添加音符，region = 光标位置，出现一个虚拟音符提示，鼠标按下添加音符
    + 右侧出现小节属性工具栏
        + 前后插入小节
        + 单小节附属符号添加
        + volta添加
    + delete快捷键删除
+ 选中音符头，进入音符头选中状态
    + 按住拖拽改变region
    + 右侧出现音符头属性工具栏
        + 单
    + delete快捷键删除
+ 选中连音线
    + 展示四个拖拽点更改连音线左右曲柄
    + 隐藏属性工具栏
    + delete快捷键删除
+ 选中其它符号
    + 隐藏属性工具栏
    + delete快捷键删除
+ 选中volta
    + 右侧出现volta属性工具栏
    + delete快捷键删除
