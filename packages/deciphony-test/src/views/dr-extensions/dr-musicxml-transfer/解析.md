**score-partwise**

+ work

    + work-title 曲谱标题

+ identification

    + creator 作者

+ part 复谱表的意思，但是musicxml的文件好像都只有一个复谱表，都是需要自己去根据美观度自动换行复谱表

    + measure:小节

        + :@

            + @_new-system? 为yes说明要换行（新的复谱表）

        + attributes

            + clef 谱号，注意如果不是复谱表第一个小节，measure内clef是第一个标签，clef在前方小节线前
                + sign 谱号 G F C 高低中 percussion TAB 打击乐谱号 吉他tab
                + line 谱号中心点落在第几条线上，这个没什么意义，谱号都是在中心的
            + divisions 每四分音符多少tick 配合note.duration使用
            + key
                + fifths 调号 1表示一个升号 -1表示一个降号，以此类推
                + mode minor|major 大调小调
            + time? 拍号 默认 4 4
                + beat
                + beat-type
            + staves 单谱表数量 这个是继承制，一般第一个小节内第一个标签会有一个，表示单谱表数量

        + note 音符

            + pitch? 音高 alter是把包括measure上，音符上等等变音符号计算好后的最终的半音偏移

                + ```
          {step:'C'.alter:0,octave:4} = midi=60 alter（-2 -1 0 1 2）半音偏移
          ```

            + staff 有值的话从1开始， 音符在复谱表内第几个单谱表上

            + stem "up"|"down"符干方向

            + duration 实际播放时值，这个依赖于divisions

            + type whole|half|quarter|eighth|16th|32nd|64th 音符时值

            + voice 音符声部，正常为1

            + rest? 和pitch互斥，表示休止符

            + accidental?  sharp flat natural double-sharp flat-flat

            + dot? 有几个附点，就写几个标签

            + beam? 符杠  begin|continue|end  number属性：1 8分音符符杠 2 16分音符符杠 3 32分音符符杠以此类推。

            + notations 附加演奏记号

                + slur? 连音线  属性type = start| stop
                + tie? 延音线  属性type = start| stop
                + ornaments？ trill-mark 颤音 mordent 回音 turn 波音  grace 经过音 等等很多
