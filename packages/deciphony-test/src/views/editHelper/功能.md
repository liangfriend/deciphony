## 五线谱编辑功能

### 持续选中的状态

+ 音符 or 休止符, 默认音符
+ 全 二分 四分 八分 16分 32分 64分 128分 默认四分
+

### 曲谱上的添加功能

+ 曲谱末尾（g-d插槽）添加复谱表按钮，始终存在
+ hover, 鼠标移动到任意曲谱元素上都会加上一个特殊的高亮filter, 移出后取消，进入进出过程有0.2s动画 完成
+ 选中小节，进入音符添加模式
    + 鼠标移动到某一范围可以添加音符，region = 光标位置，出现一个虚拟音符提示，鼠标按下添加音符
    + 右侧出现小节属性工具栏
        + 前后插入小节
        + 反复符号添加
        + 小节线添加/修改
        + 前置谱号添加/修改
        + 前置调号添加/修改
        + 前置拍号添加/修改
        + volta添加
    + delete快捷键删除
+ 选中音符头，进入音符头选中状态
    + 按住拖拽改变region
    + 右侧出现音符头属性工具栏
        + 时值修改
        + 符杠属性（全连/右连/无）
        + 变音符号添加/修改
        + 附点添加/修改
    + delete快捷键删除
+ 选中连音线
    + 展示四个拖拽点更改连音线左右曲柄
    + 隐藏属性工具栏
    + delete快捷键删除
+ 选中其它符号
    + 隐藏属性工具栏
    + delete快捷键删除
+ 选中volta
    + 展示左右拖拽点，拖拽更换小节覆盖范围
    + 右侧出现volta属性工具栏
        + 设置反复值
        + 设置文案
    + delete快捷键删除

简谱编辑功能（numberNotation/）

+ 与五线谱共用：小节选中、插槽按钮、volta/slur 拖拽、删除快捷键
+ 简谱专有：上下拖拽更改唱名 1–7；添加节奏音符 X
+ 属性栏无符干方向；时值/减时线连接在 NoteNumber 层级
