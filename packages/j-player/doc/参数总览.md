# 参数总览

使用任何播放器前，需要先调用 `startJPlayer()` 激活音频上下文。

```js
import { startJPlayer } from '@jsh/j-player';
await startJPlayer();
```

---

## 全局函数

| 函数              | 说明                                        |
| ----------------- | ------------------------------------------- |
| `startJPlayer()`  | 激活全局 AudioContext，所有播放器实例共享   |
| `closeJPlayer()`  | 销毁全局 AudioContext                       |
| `activeContext()` | 恢复被浏览器挂起的 AudioContext（`resume`） |

---

## APlayer — 音频文件播放器

适用于播放完整音频文件（mp3、wav 等）。

### 属性

| 属性       | 类型      | 读写      | 默认值  | 说明                                                                     |
| ---------- | --------- | --------- | ------- | ------------------------------------------------------------------------ |
| `volume`   | `number`  | get / set | `1`     | 音量，0\~1                                                               |
| `rate`     | `number`  | get / set | `1`     | 播放速率，播放中可实时修改                                               |
| `loop`     | `boolean` | get / set | `false` | 是否循环播放                                                             |
| `current`  | `number`  | get / set | `0`     | 当前播放位置（秒）。**set 仅在非 playing 状态可用**，播放中请用 `seek()` |
| `duration` | `number`  | get       | `0`     | 音频总时长（秒），`setAudio` 后可用                                      |

### 回调

| 回调         | 类型                                          | 说明                                           |
| ------------ | --------------------------------------------- | ---------------------------------------------- |
| `onProgress` | `(current: number, duration: number) => void` | 播放过程中每帧触发，返回当前时间和总时长（秒） |
| `onEnd`      | `() => void`                                  | 自然播放结束时触发（手动 `stop` 不触发）       |

### 方法

| 方法              | 参数                                   | 返回值          | 说明                                          |
| ----------------- | -------------------------------------- | --------------- | --------------------------------------------- |
| `setAudio(input)` | `string \| AudioBuffer \| ArrayBuffer` | `Promise<void>` | 加载音频。支持 URL、ArrayBuffer、AudioBuffer  |
| `play()`          | —                                      | `Promise<void>` | 开始/恢复播放                                 |
| `pause()`         | —                                      | `void`          | 暂停播放                                      |
| `stop()`          | —                                      | `void`          | 停止并重置到起始位置                          |
| `seek(value)`     | `number`（秒）                         | `Promise<void>` | 跳转到指定位置，播放中和暂停/停止状态均可调用 |
| `dispose()`       | —                                      | `void`          | 销毁实例，释放资源和回调                      |

## NPlayer — 音符序列播放器

适用于按 MIDI 序列播放合成音符，支持多音色、单音符触发。

### 时值单位

时值以 256 分音符为最小粒度：

| 值  | 时值含义  |
| --- | --------- |
| 1   | 256分音符 |
| 2   | 128分音符 |
| 4   | 64分音符  |
| 8   | 32分音符  |
| 16  | 16分音符  |
| 32  | 8分音符   |
| 64  | 4分音符   |
| 128 | 2分音符   |
| 256 | 全音符    |

### 属性

| 属性       | 类型      | 读写      | 默认值  | 说明                                      |
| ---------- | --------- | --------- | ------- | ----------------------------------------- |
| `volume`   | `number`  | get / set | `1`     | 音量，0\~1                                |
| `bpm`      | `number`  | get / set | `120`   | 每分钟拍数，播放中可实时修改              |
| `beatUnit` | `number`  | get / set | `4`     | 几分音符为一拍（拍号分母），用于 bpm 换算 |
| `rate`     | `number`  | get / set | `1`     | 播放速率，播放中可实时修改                |
| `loop`     | `boolean` | get / set | `false` | 是否循环播放                              |

### 回调

| 回调         | 类型                                    | 说明                                                                                 |
| ------------ | --------------------------------------- | ------------------------------------------------------------------------------------ |
| `onProgress` | `(progress: number, data: any) => void` | 每个音符播放时触发。`progress` 为当前播放秒数，`data` 为 SequenceItem 中的自定义数据 |
| `onEnd`      | `() => void`                            | 序列播放结束时触发（`loop = false` 时）                                              |

### 方法

| 方法                            | 参数                                   | 返回值          | 说明                                                                                        |
| ------------------------------- | -------------------------------------- | --------------- | ------------------------------------------------------------------------------------------- |
| `addToneColor(name, toneColor)` | `name: string`, `toneColor: ToneColor` | `Promise<void>` | 注册音色。`ToneColor = Record<number, string>`，key 为 MIDI 号，value 为音频 URL 或 dataURL |
| `setSequence(sequence)`         | `Sequence`                             | `void`          | 设置播放序列                                                                                |
| `play()`                        | —                                      | `Promise<void>` | 开始/恢复播放                                                                               |
| `pause()`                       | —                                      | `void`          | 暂停播放                                                                                    |
| `stop()`                        | —                                      | `void`          | 停止并重置                                                                                  |
| `trigger(param)`                | 见下方                                 | `string`        | 立即播放单个音符，返回 id                                                                   |
| `release(param)`                | `{ id: string }`                       | `void`          | 停止指定的单音符播放                                                                        |
| `releaseAll()`                  | —                                      | `void`          | 停止所有单音符播放                                                                          |
| `dispose()`                     | —                                      | `void`          | 销毁实例，释放全部资源                                                                      |

### trigger 参数

| 字段        | 类型     | 必填 | 说明                                     |
| ----------- | -------- | ---- | ---------------------------------------- |
| `id`        | `string` | 否   | 自定义 id，不传则自动生成 UUID           |
| `midi`      | `number` | 是   | MIDI 音高                                |
| `toneColor` | `string` | 是   | 音色名称（需已通过 `addToneColor` 注册） |
| `volume`    | `number` | 否   | 音量，默认 1                             |
| `offset`    | `number` | 否   | 延迟播放秒数，默认 0                     |
| `duration`  | `number` | 否   | 持续秒数，不传则播完整个 buffer          |

### Sequence 数据结构

```ts
type Sequence = SequenceItem[][];
```

每个子数组内的音符在同一拍点开始播放，下一组的开始时间 = 上一组开始时间 + 上一组内最长音符时值。

**SequenceItem：**

| 字段        | 类型      | 必填 | 说明                                   |
| ----------- | --------- | ---- | -------------------------------------- |
| `midi`      | `number`  | 是   | MIDI 音高，-1 为休止符                 |
| `duration`  | `Unit256` | 是   | 播放时值                               |
| `toneColor` | `string`  | 是   | 音色名称                               |
| `velocity`  | `number`  | 否   | 力度系数（0\~1），默认 1               |
| `data`      | `any`     | 否   | 自定义数据，会透传到 `onProgress` 回调 |
| `offset`    | `Unit256` | 否   | 延时播放时值                           |

## MPlayer — 节拍器播放器

适用于节拍器场景，按拍号自动生成强弱拍序列。

### 属性

| 属性             | 类型            | 读写      | 默认值  | 说明                                                              |
| ---------------- | --------------- | --------- | ------- | ----------------------------------------------------------------- |
| `volume`         | `number`        | get / set | `1`     | 音量，0\~1                                                        |
| `bpm`            | `number`        | get / set | `120`   | 每分钟拍数，播放中可实时修改                                      |
| `beatUnit`       | `number`        | get / set | `4`     | 几分音符为一拍                                                    |
| `rate`           | `number`        | get / set | `1`     | 播放速率，播放中可实时修改                                        |
| `loop`           | `boolean`       | get / set | `false` | 是否循环播放                                                      |
| `timeSignature`  | `TimeSignature` | get / set | `'4/4'` | 拍号，格式为 `"分子/分母"`。支持 1/2\~16/16。播放中设置会自动停止 |
| `metronomeColor` | `string`        | get / set | `''`    | 当前使用的节拍器音色名称                                          |

### 回调

| 回调         | 类型                                                      | 说明                                                                        |
| ------------ | --------------------------------------------------------- | --------------------------------------------------------------------------- |
| `onProgress` | `(progress: number, data: { beatIndex: number }) => void` | 每拍触发。`progress` 为当前秒数，`data.beatIndex` 为当前第几拍（从 0 开始） |
| `onEnd`      | `() => void`                                              | 一小节播放结束时触发（`loop = false` 时）                                   |

### 方法

| 方法                             | 参数                                    | 返回值          | 说明               |
| -------------------------------- | --------------------------------------- | --------------- | ------------------ |
| `addMetronomeColor(name, color)` | `name: string`, `color: MetronomeColor` | `Promise<void>` | 注册节拍器音色     |
| `play()`                         | —                                       | `Promise<void>` | 开始/恢复播放      |
| `pause()`                        | —                                       | `void`          | 暂停播放           |
| `stop()`                         | —                                       | `void`          | 停止并重置         |
| `dispose()`                      | —                                       | `void`          | 销毁实例，释放资源 |

### MetronomeColor 数据结构

```ts
type MetronomeColor = Record<Intensity, string>;
type Intensity = 'strong' | 'weak' | 'secondary';
```

| 字段        | 说明                       |
| ----------- | -------------------------- |
| `strong`    | 强拍音频（URL 或 dataURL） |
| `weak`      | 弱拍音频                   |
| `secondary` | 次强拍音频                 |
